# ⭐ Task 2 – User Review Dashboard (Fynd Internship Assessment)

This project contains two Streamlit dashboards built for Task 2 of the Fynd Take-Home Assignment.

## ✔ What the Project Does
- Users can submit a **rating (1–5)** and a **text review**.
- The app generates an **AI summary** using the Gemini Flash Lite model.
- All submissions are saved locally in a CSV file.
- Admin can view all entries and see analytics.

## ✔ Files Included
- `user_dashboard.py` — User interface for adding reviews  
- `admin_dashboard.py` — Admin panel to view data  
- `utils.py` — Helper functions for saving and loading entries  
- `database.csv` — Stored reviews (auto-created)
